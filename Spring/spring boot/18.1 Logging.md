# Logging levels
1. trace
2. debug
3. info - default we wont get trace and debug by default
4. warn
5. error
6. off

for better logs at non prod level, to make debug level logs work
in the application.properties
```yaml
logging:
	level:
		root: TRACE
```
trace prints literally everything

we have equivalent methods for the log levels
```java
logger.trace();
logger.debug();
logger.info();
logger.warn();
logger.error();
```
## code
import from SL4J
```java
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;

private static final Logger logger = LoggerFactory.getLogger(WeatherController.class);

@GetMapping("/hello")
public String hi(@RequestHeader("Authorization") String authToken){
	String username = JWTUtils.getUsername(authToken);
	logger.info("Received request for /hello from {}", username);
	return "Hello World";
}
```

# writing logs to a file
to write logs to a file, to the application.yml, add
```yaml
logging:
	file:
		name: logs/app.log
```
and this appends to the file if it already exists
