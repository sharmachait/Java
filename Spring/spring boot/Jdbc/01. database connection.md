# By default spring boot provides us a connection pool of min and max = 10 db connections

we can customize it with
```properties
spring.datasource.hikari.maximum-pool-size=10
spring.datasource.hikari.minimum-idle=5
```

or we can manually inject the bean for it
```java
@Configuration
public class AppConfig {
	@Bean
	public DataSource dataSource(){
		HikariDataSource ds = new HikariDataSource();
		ds.setDriverClassName("");
		ds.setJdbcUrl("");
		ds.setUsername("");
		ds.setPassword("");
		return ds;
	}
}
```

## DriverManagerDataSource

- **Not a Connection Pool**: Creates a new database connection for every request.
## HikariDataSource

- **Real Connection Pool**: Part of the HikariCP library, `HikariDataSource` provides a high-performance, lightweight JDBC connection pool
